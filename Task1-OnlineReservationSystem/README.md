# Online Reservation System

## Oasis Infobyte Java Internship (OIBSIP)

### Developed By
Mohit Mathur

## Description
This is a Java console-based Online Reservation System developed as part of the Oasis Infobyte Java Internship Program.

## Features
- User Login
- Ticket Reservation
- Automatic PNR Generation
- View Reservation
- Ticket Cancellation

## Technologies Used
- Java
- Object-Oriented Programming
- HashMap
- Scanner

## How to Run

Compile:
```bash
javac OnlineReservationSystem.java
```

Run:
```bash
java OnlineReservationSystem
```
